Лабораторная №3
Инструкции
Реализовать новую программу или модернизировать программу из Лабораторной №2

Создать параллельный поток. (наприпер классом Thread из библиотеки threading).

Использовать коллекцию (например список [ ]) для хранения и динамического 
добавления/удаления элементов. Т.е. нужно чтобы элементы появлялись и удалялись 
в процессе работы программы. Работа c коллекцией должа вестьсиь в нескольких потоках!

Добавить отрисовку изображений с задаваемым поворотом (либо переопределив методы 
paintEvent() у компонентов, либо используя класс QTransform из PyQt)

Можете сделать любое интерактивное приложение, - главное это наличие поворота 
изображения, параллельного потока и коллекции.
